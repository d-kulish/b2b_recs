Pricing

Recs Studio is offered as a managed service on Google Cloud. Every customer gets an isolated GCP project with dedicated infrastructure. Pricing has two components: a fixed Recs Studio management fee and the underlying Google Cloud infrastructure, which we pass through at cost.

What's Included in Infrastructure Costs?

Each customer runs in a dedicated Google Cloud project. Typical monthly infrastructure for a single production model starts around €700 and includes Cloud Run (app and serving), Cloud SQL, BigQuery, Cloud Storage, Cloud Scheduler, Vertex AI training, and Cloud NAT with a static IP. We invoice GCP costs at cost; there is no markup.

Custom Builds and Marketplace Migration

If you need a fully tailored implementation before the managed Marketplace listing is available, we offer custom-build engagements. Existing custom-build customers continue to receive full support and will be migrated to the Marketplace billing model once the listing goes live.
