How to use with Resend (free plan):

1. Pick a signature option above and copy the HTML between the SIGNATURE comments.
2. Host the logo at a public URL (e.g. upload recs_logo.png to your website, an S3 bucket, or use imgur.com for quick testing). Update the src attribute.
3. Build your email HTML by combining a body + the signature footer.
4. Send via the Resend API:

import resend resend.api_key = "re_YOUR_API_KEY" html = open("your_email.html").read() resend.Emails.send({ "from": "Dmytro Kulish <dkulish@recs.studio>", "to": ["recipient@example.com"], "subject": "Your subject", "html": html, })
Font note: The template uses Inter from Google Fonts. It renders natively in Apple Mail, iOS Mail, and Outlook.com. Gmail and Outlook desktop ignore web fonts and fall back to their default sans-serif (which still looks clean).
